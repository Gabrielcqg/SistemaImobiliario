# Radar Globe Assets

Place public domain assets here:

1) `globe/landmask.png`
- Equirectangular land mask (land = white, ocean = black).
- Suggested source: Natural Earth land mask (public domain).

2) `ne_110m_coastline.geojson` (opcional)
- Coastline lines (public domain), e.g. Natural Earth.

The UI will render without these files, but continents/coastlines will be subtler.
